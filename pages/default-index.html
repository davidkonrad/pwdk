Gawain default index

Use this document as a way to quickly start a new project. You have several options for customizing this page right away.

Change the design

  • Go to admin-routes/system.routes/ and choose a different template for / (this route)
  • You can go to admin-bundles and include a framework and / or various libraries
  • Or you can do both

Change the content

  • Go to admin-routes/system.routes/ and choose a different include file for / (this route). Any filetype is acceptable, you can place them anywhere in the filesystem, as long as the path are accesible for the www-root
  • Or simply edit pages/default-index.html (this page) and feel free to do the same with templates/default/default.css.

Add new pages

  • Go to admin-routes and create a new .routes file
  • For each page / piece of content you want to include, add one or more routes and provide the path for the include file.
